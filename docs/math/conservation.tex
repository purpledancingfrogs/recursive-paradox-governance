\\subsection{Definitions}

\\subsection{Theorem 1 (Non-Annihilation)}

\\subsection{Theorem 2 (Bounded Coherence)}
0 \\le \\Xi(t) \\le 1 \\; \\forall t.

\\subsection{Theorem 3 (Horizon Preservation)}
